# Configuração do Arch Linux, mais download de aplicativos para quem tiver duvidas.

## Download YAY.

[wiki.arch-linux](https://aur.archlinux.org/packages/yay/)

## Instalação

```sh
pacman -S base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
```
